fn target(&mut self, t: Target) -> &mut Self

Adds a target to generate code for.

The first target that's added replaces the default list. Every subsequent target will be added to this new list of targets. Only one target per ISA may be selected. For example, you may have the simultaneous choises of [ Sse2_i32x4, Sse4_i32x8 ], but not [ Sse2_i32x4, Sse2_i32x8 ].

Default value: [ Sse2, Sse4, Avx1, Avx1_1, Avx2 ]
